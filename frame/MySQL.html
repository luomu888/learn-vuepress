    # 基础篇

    数据库管理系统：Navicat、sqlyog、datagrip等

    SQL分类：

    • DDL：data definition（定义）数据库，表，字段
    • DML：data manipulation（操作），增删改
    • DQL：data query（查询），查
    • DCL：data control（控制），数据库访问权限，创建数据库用户

    select database()：查询当前使用哪个数据库

    desc table：查看表的详细信息

    字段类型blob：二进制数据，视频等

    in(1,3,5,6)：在列表中的值选一个，注意不是范围

    where后面不能使用聚合函数，但是select 前面可以使用，而having（分组的过滤条件）可以

    like：模糊查询，_代表一个字，%代表很多字，例 _ _ _ 姓名为三个字的员工，张%，姓名以张开头，张三，张三丰等
