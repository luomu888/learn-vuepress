    # 基本介绍

    采用级层式的==树状目录结构==，==最上层是根目录”/“==

    记住：==在Linux世界里，一切皆文件！！！==

    # 具体的目录结构

    根是：/

    1. /bin[常用]（/usr/bin、/usr/local/bin）：==存放经常使用的命令==
    2. /sbin（/usr/sbin、/usr/local/sbin）：s是Super User的意思，即==系统管理员使用的系统管理程序==
    3. /home[常用]：存放==普通用户的主目录==
      • 添加用户指令：==useradd== 用户名（tom）
      • 删除用户指令：==userdel -r== tom(用户名)
    4. /root[常用]：系统管理员的==用户主目录==
    5. /lin：系统开机所需基本的==动态连接共享库==，类似Windows里的==DLL==（ 动态链接库（dynamic link library））文件，使用的==应用程序都需要这个共享库==
    6. /lost+found：==系统非法关机后==，存放的一些文件
    7. /etc[常用]：==配置文件和子目录==
      • 例：安装MySQL，my.conf
    8. /usr：很多应用程序和文件，==类似Windows下的program files目录==
    9. /boot[常用]：连接文件以及==镜像文件==
    10. /proc[不能动]：虚拟目录，==系统内存的映射==
    11. /srv[不能动]：==存放一些服务启动后需要的数据==
    12. /tmp：==存放临时文件==
    13. /dev：==类似window的设备管理器，==不过是所有硬件以文件的形式存储，即==在Linux世界里，一切皆文件！！！==
    14. /media[常用]==：识别设备，例U盘、光驱等，识别后挂载到这里==
    15. /mnt[常用]：临时挂载别的文件系统，即将==外部的（window）挂载在这个目录下==
    16. /opt：==安装软件时，先放到这==
    17. **/usr/local[**常用]
    18. /var[常用]：==日志文件，经常修改的目录==
    19. /selinux[security-enhanced linux]：安全增强、==控制程序只能访问特定文件==

    # Linux组基本介绍

    # 组的创建

    • 创建一个用户，为他分配组，不分配就是默认和用户名一样

    # 改变用户所在组

    # 权限的基本介绍

    0~9

    • 第零位：d - l c b
      • l（link）是超链接，即快捷方式
      • d（directory）是目录
      • c是==字符设备==文件，鼠标，键盘
      • b是==块设备==，硬盘
    • 1~3所有者权限，u(user)
    • 4~6所属组权限，g(group)
    • 7~9其它用户权限，o(other)
    • 第一个root是用户，第二个root是组，4069是文件的大小

    # rwx权限详解

      • -W：显示需要备份的文件以及最后一次备份的层级，时间，日期
